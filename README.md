## Compiling
teh value of `-DRETRO_GAMETYPE` can be
- `0` - Sonic 1 Forever
- `1` - Sonic 2 Absolute
- `2` - Sonic CD Timeless

then just run
```
cmake -B build -DRETRO_GAMETYPE=0
cmake --build build --config release
```
